## 步驟

1. **開新的 Notebook**，把課本或習作 PDF 匯入資料來源
2. **在對話區下指令**（範例在下方，替換 [方括號] 內容）
3. **逐題查證**：每題都有來源編號，點開對照課本原文
4. **保存**：把題目「加入記事」，或複製到 Google 文件編輯後印製；想做線上測驗可貼進 edcafe（QR code 登入、自動記錄作答）
