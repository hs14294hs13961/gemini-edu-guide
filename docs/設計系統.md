# 設計系統 — Apple 風格配色

來源：Shao-Chun 的 Claude Design「Apple Design System」專案（2026-07-09 抽取）
原則：白底為主、單一藍色強調、無多餘陰影與裝飾——「介面退位，內容當主角」。

## 色彩 Token

### 強調色（唯一互動色，所有連結/按鈕/焦點都用它）

| Token | 色碼 | 用途 |
|-------|------|------|
| `primary` | `#0066cc` | Action Blue：連結、主按鈕 |
| `primary-focus` | `#0071e3` | Focus Blue：hover/focus 狀態 |
| `primary-on-dark` | `#2997ff` | Sky Blue：深色底上的連結（僅深色區用） |

### 淺色表面（頁面主體）

| Token | 色碼 | 用途 |
|-------|------|------|
| `canvas` | `#ffffff` | 主背景 |
| `parchment` | `#f5f5f7` | 區塊交替背景（用背景色變化取代分隔線） |
| `pearl` | `#fafafc` | 卡片內淺灰底 |

### 深色表面（頁尾、強調區塊）

| Token | 色碼 | 用途 |
|-------|------|------|
| `surface-tile-1` | `#272729` | 深色區塊主底 |
| `surface-tile-2` | `#2a2a2c` | 深色區塊亮一階 |
| `surface-tile-3` | `#252527` | 深色區塊暗一階 |
| `surface-black` | `#000000` | 導覽列 |

### 文字

| Token | 色碼 | 用途 |
|-------|------|------|
| `ink` | `#1d1d1f` | 標題與正文 |
| `ink-muted-80` | `#333333` | 次要文字 |
| `ink-muted-48` | `#7a7a7a` | 輔助說明、標籤 |

### 邊框

| Token | 色碼 | 用途 |
|-------|------|------|
| `hairline` | `#e0e0e0` | 卡片、chip 邊框（1px） |
| `divider-soft` | `#f0f0f0` | 軟分隔線 |

## 規則（照抄設計系統的紀律）

- 圓角只有兩套文法：`9999px` 膠囊（按鈕、搜尋框）；`18px` 卡片、`8px` 工具鈕、`11px` 小膠囊。不混用
- 卡片：白底＋1px hairline 邊框＋18px 圓角＋24px 內距，**不加陰影**
- 全站只有一種陰影（產品圖用），卡片與按鈕永不用陰影
- 區塊分隔靠背景色交替（白 ↔ parchment ↔ 深色），不用邊框線
- 間距：8px 基準；區塊上下 80px、卡片內距 24px
- 動效極簡：按下 `scale(0.95)`，無進場動畫；sticky 導覽用毛玻璃 `blur(20px) saturate(180%)`
- 字重階梯 300/400/600/700（沒有 500）；標題負字距

## 字體（台灣中文調整）

原設計用 SF Pro，中文需調整：

```css
font-family: -apple-system, "SF Pro TC", "PingFang TC",
             "Noto Sans TC", Inter, "Microsoft JhengHei", sans-serif;
```

- Apple 裝置自動用系統字體（英文 SF Pro、中文蘋方）
- Windows/Android 載入 Noto Sans TC（Google Fonts 免費）
- 正文 17px（不是 16），行高照 Apple 節奏

## 教育網站的在地調整

- 對比度：`ink-muted-48`（#7a7a7a）在白底約 4.4:1，僅用於大字或標籤，正文一律 `ink`
- 難度標籤配色到 Phase 1 再定（入門/進階/挑戰需要可辨識的三色，但要克制——維持單一強調色原則，考慮用灰階＋圖示或藍色深淺）
- 版權：色彩數值與間距規則不受著作權保護，可安心使用；不使用 Apple 商標、產品圖與 SF 字體檔
